Mall Management System - Item Module

This project is an Item Module developed as part of a Mall Management System. It focuses on managing item details, product information, and inventory-related records within the mall.

Features

- Add item details
- View item information
- Update item records
- Delete item records
- Manage product inventory efficiently
- Track item-related data within the mall

Technologies Used

- Java
- Maven
- MySQL
- JDBC / Spring Boot

Project Objective

The objective of this project is to simplify the management of item information and demonstrate the implementation of CRUD (Create, Read, Update, Delete) operations using Java and database technologies.
